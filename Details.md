# Main features
1.LTPS calculator
2.Classes to Go

## LTPS Calculator
[L-Lecture T-Tutorial P-Practical S-Skill]

### Numeric mode[already made]
- User input: classes attended and conducted in each of LTPS
- gives final/Total attendance on parameter of LTPS:100%:100%:50%:25% 

### Percentage mode
- User input: percentage of classes attended and conducted in each of LTPS
- gives final/Total attendance on parameter of LTPS:100%:100%:50%:25% 

## Classes to Go
- User input: 
  - classes attended and conducted in each of LTPS
  - number of classes left in each of LTPS 
  - desired attendance percentage
- gives number of classes to attend out of remaining classes in each of LTPS such that final attendance is >= desired attendance percentage
